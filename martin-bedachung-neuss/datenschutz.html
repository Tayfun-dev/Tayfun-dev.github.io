Hosting und Bereitstellung der Website

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Website können technisch erforderliche Daten verarbeitet werden, damit die Inhalte im Browser angezeigt werden können.

Dazu können insbesondere IP-Adresse, Datum und Uhrzeit des Abrufs, aufgerufene Dateien, Browserinformationen und weitere technisch notwendige Verbindungsdaten gehören.

Domain

Die Website ist unter der Domain martin-bedachung.de erreichbar. Bei der Nutzung der Domain können technisch erforderliche Verbindungsdaten verarbeitet werden, damit die Website erreichbar ist.

Falls Sie hierzu genaue Angaben zum Domain-Anbieter ergänzen möchten, können diese an dieser Stelle später noch konkretisiert werden.

Kontaktformular

Wenn Sie das Kontaktformular nutzen, werden die von Ihnen eingegebenen Daten zur Bearbeitung Ihrer Anfrage verarbeitet. Dazu gehören insbesondere Ihr Vorname, Nachname, Ihre Telefonnummer, die gewählte Leistung und Ihre Nachricht.

Das Formular wird über Formspree bereitgestellt. Beim Absenden werden Ihre Angaben an Formspree übermittelt, damit Ihre Anfrage technisch verarbeitet und an den Website-Betreiber weitergeleitet werden kann.

Bitte geben Sie im Formular nur die Daten an, die für Ihre Anfrage erforderlich sind.

Telefon und WhatsApp

Auf dieser Website finden Sie klickbare Telefon- und WhatsApp-Links.

Wenn Sie auf den Telefon-Link klicken, wird eine Verbindung zu Ihrer Telefonfunktion oder Telefon-App hergestellt.

Wenn Sie auf den WhatsApp-Link klicken, wird eine Verbindung zu WhatsApp hergestellt. Dabei können Daten an WhatsApp oder verbundene Dienste übermittelt werden. Auf diese Datenverarbeitung hat der Website-Betreiber keinen Einfluss.

Cookies, Analyse und Werbung

Auf dieser Website werden nach aktuellem Stand keine eigenen Analyse-, Tracking- oder Werbedienste eingesetzt.
